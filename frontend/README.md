# TaskPulse Frontend

A React task manager UI built with Vite.

## Features

- Add tasks via input field (press Enter or click Add)
- Mark tasks as complete/undo with a toggle button
- Delete tasks
- Fetches from and persists to the backend at `http://localhost:3000`

## Development

```bash
npm install
npm run dev
```

Requires the backend running on port 3000.
